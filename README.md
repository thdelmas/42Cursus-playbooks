### Installation

```sh
git clone https://github.com/thdelmas/42Cursus-Playbooks # Fetch source files
cd 42Cursus-playbooks
python3 -m venv venv
source venv/bin/activate 
pip install -r requirements.txt
```

### Usage

```sh
cd 42Cursus-playbooks
source venv/bin/activate
ansible -m ping all
ansible-playbook site.yml
```
